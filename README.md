## API Reference

#### Get all customers

```http
  GET /api/data
```

#### Create customer

```http
  POST /api/data/
```

#### Get customer

```http
  GET /api/data/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of customer to fetch |

#### Delete customer

```http
  DELETE /api/data/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of customer to delete |

#### Update customer

```http
  PUT /api/data/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to update |
